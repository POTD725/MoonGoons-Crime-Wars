## In-game chat

Press **F8** during RTS Operations, Free Roam, or a LAN co-op recon session.

- `Enter` opens chat when it is closed.
- `Esc` closes chat.
- LAN messages are sent through the host and shared with the party.
- Offline missions keep messages local, including system notices.

Chat commands:

- `/help` — chat command list
- `/map` — current battlefield and resource level
- `/roll` — roll 1–100 for the party
- `/clear` — clear the current local chat display

The LAN lobby retains its own roster/chat panel, while F8 is the persistent in-game chat once the shared recon map is running.
