# MoonGoons Core PNG Asset Manifest

Install the companion `MoonGoons_Core_PNG_Assets.zip` into this folder before running the game. The live renderer prefers these PNGs and only falls back to the legacy procedural art when a file is missing.

## Structures
- `structures/command_nexus.png` -> `nexus`
- `structures/tactical_armory.png` -> `armory`
- `structures/machine_shop.png` -> `machine_shop`

## Troops
- `troops/builder_drone.png` -> `drone`
- `troops/patrol_deputy.png` -> `deputy`
- `troops/shield_deputy.png` -> `shield`

## Defenses
- `defenses/sentry_turret.png` -> `sentry_turret`
- `defenses/pulse_cannon.png` -> `pulse_cannon`

## Resources
- `resources/ore_deposit.png` -> `ore`
- `resources/evidence_cache.png` -> `evidence`

## Environment
- `environment/cargo_crate.png`
- `environment/wrecked_shuttle.png`
- `environment/cargo_wall.png`
- `environment/crater.png`

Every asset is an isolated 512Ă—512 RGBA PNG with transparent background and no text baked into the image.
