## Output-folder status

The original MoonGoons export configuration writes Windows builds to `builds/windows/MoonGoons.exe`. That is an export output target, not a committed asset package. No committed executable or output-art bundle was found at that location during this import pass.

The Dark Side repository documents three JPEG web-art assets, but the actual files were not retrievable at the documented repository paths during this import pass. They are therefore not represented as imported files here.
